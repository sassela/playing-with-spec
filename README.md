.com/blog/2016/05/29/one-fish-spec-fish/), [Arne Brasseur's LambdaIsland video]
(https://lambdaisland.com/episodes/clojure-spec) and [Stuart Halloway's Leverage talk](https://www.youtube.com/watch?v=nqY4nUMfus8.
